# Quest Game

## Описание проекта

Quest Game ? это консольная игра на языке C++, в которой игрок исследует локации, решает головоломки, открывает сундуки и использует зелья для восстановления здоровья.

Проект разработан с использованием объектно-ориентированного программирования (ООП). Каждый элемент игры реализован в виде отдельного класса.

## Возможности игры

* Перемещение между локациями.
* Решение головоломок.
* Открытие сундуков.
* Использование зелий.
* Отображение информации о локациях.
* Запуск и управление игровым процессом через класс Game.

## Структура проекта

### Character

Класс игрока. Хранит информацию о здоровье и текущем местоположении персонажа.

### Chest

Класс сундука. Содержит информацию о предметах и позволяет открывать сундуки.

### Potion

Класс зелья. Используется для восстановления здоровья игрока.

### Location

Класс игровой локации. Хранит название, описание и информацию о наличии сундука.

### Puzzle

Класс головоломки. Содержит вопрос, сложность и проверку ответа.
### Game
Главный класс игры. Управляет запуском и выполнением игрового процесса.

## Используемые технологии
* Язык программирования: C++
* Компилятор: g++
* Среда разработки: Visual Studio Code
* Операционная система: Ubuntu (WSL)
## Сборка проекта
Для компиляции необходимо выполнить команду:
g++ *.cpp -o questgame
Для запуска программы:
./questgame
