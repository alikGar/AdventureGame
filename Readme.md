# AdventureGames

## Описание проекта

AdventureGames — это консольная приключенческая игра, разработанная на языке C++. Игроку предстоит пройти через несколько локаций, решить загадки, найти магические зелья и добраться до финальной точки путешествия.

Основная цель игры — пройти все испытания, сохранив как можно больше здоровья. Неправильные ответы на загадки уменьшают здоровье персонажа, а найденные зелья помогают его восстановить.

Проект разработан с использованием объектно-ориентированного программирования (ООП). Каждый элемент игры представлен отдельным классом, что делает код более удобным для поддержки и расширения.

---

## Игровой процесс

Игрок начинает приключение с 50 единицами здоровья.

### Этап 1. Forest Entrance

Начальная локация игры. Игрок появляется у входа в таинственный лес и может начать путешествие.

### Этап 2. Hunter Hut

Игрок находит хижину охотника и открывает деревянный сундук.

Награда:

* Forest Herb (+20 HP)

После использования зелья здоровье игрока увеличивается.

### Этап 3. Dark Cave

Игрок попадает в тёмную пещеру и сталкивается с первыми загадками.

Если ответ неверный:

* игрок получает урон;
* здоровье уменьшается.

### Этап 4. Ancient Temple

В древнем храме игрок проходит ещё несколько испытаний и находит древний сундук.

Награда:

* Ancient Elixir (+50 HP)

### Этап 5. Treasure Gate

Финальная локация игры.

Если игрок успешно дошёл до конца маршрута, игра завершается сообщением о победе.

## Структура проекта

### Character

Класс игрока.

Функции:

* хранение здоровья персонажа;
* хранение текущей локации;
* получение урона;
* работа с инвентарём.

### Chest

Класс сундука.

Функции:

* хранение предметов;
* открытие сундука;
* выдача награды игроку.

### Potion

Класс зелья.

Функции:

* хранение названия зелья;
* хранение количества восстанавливаемого здоровья;
* использование зелья.

### Location

Класс игровой локации.

Функции:

* хранение названия локации;
* отображение карты;
* отображение описания локации.

### Puzzle

Класс загадки.

Функции:

* хранение вопроса;
* хранение правильного ответа;
* проверка ответа игрока;
* нанесение урона при ошибке.

### SaveSystem

Класс сохранения игры.

Функции:

* сохранение игрового прогресса;
* загрузка сохранённой игры.

### Game

Главный класс проекта.

Функции:

* запуск игры;
* управление игровым процессом;
* перемещение между локациями;
* создание загадок;
* работа с сундуками и наградами;
* завершение игры.

## Сборка проекта
Универсальный метод с командой make:

```bash
make
```

Для запуска игры:

```bash
./AdventureGames
```
 Для Windows:

```bash
g++ -Wall -std=c++11 Character.cpp Chest.cpp Game.cpp Location.cpp Potion.cpp Puzzle.cpp Savesystem.cpp main.cpp -o AdventureGames.exe ./AdventureGames.exe
```
