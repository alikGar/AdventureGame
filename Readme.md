# AdventureGames

## Описание проекта

AdventureGames ? это консольная приключенческая игра на языке C++, в которой игрок исследует различные локации, решает загадки, находит зелья и пытается добраться до финального выхода.

Проект разработан с использованием объектно-ориентированного программирования (ООП). Каждый элемент игры реализован в виде отдельного класса.

## Возможности игры

* Перемещение между локациями.
* Решение загадок и логических задач.
* Получение зелий для восстановления здоровья.
* Система здоровья персонажа.
* Инвентарь игрока.
* Отображение карт локаций с помощью ASCII-графики.
* Сохранение и загрузка игры.
* Управление игровым процессом через класс Game.

## Структура проекта

### Character

Класс игрока. Хранит информацию о здоровье, текущем местоположении и инвентаре персонажа.

### Chest

Класс сундука. Используется для хранения игровых предметов и наград.

### Potion

Класс зелья. Позволяет восстанавливать здоровье игрока.

### Location

Класс игровой локации. Содержит название, описание, карту и информацию о наличии сундука.

### Puzzle

Класс загадки. Содержит вопрос, правильный ответ и проверку ответа игрока.

### SaveSystem

Класс сохранения. Отвечает за сохранение и загрузку игрового прогресса.

### Game

Главный класс игры. Управляет запуском игры, перемещением между локациями, загадками и игровым процессом.

## Игровые локации

* Forest Entrance ? вход в таинственный лес.
* Hunter Hut ? хижина охотника с первым сундуком.
* Dark Cave ? тёмная пещера с загадками.
* Ancient Temple ? древний храм с наградами.
* Treasure Gate ? финальная локация игры.

## Загадки игры

Во время прохождения игры игроку необходимо решить несколько загадок. Неправильный ответ уменьшает количество здоровья персонажа.

### Загадка 1

Вопрос:

You have 3 apples. You take 2 apples. How many apples do you have?

Правильный ответ:

```text
2
```
### Загадка 2

Вопрос:

What is heavier: 1 kilogram of iron or 1 kilogram of cotton?

Правильный ответ:

```text
equal
```
### Загадка 3

Вопрос:
If 2 cats catch 2 mice in 2 minutes, how many cats are needed to catch 100 mice in 100 minutes?

Правильный ответ:
```text
2
```
### Загадка 4

Вопрос:
What can be broken without touching it?

Правильный ответ:

```text
promise
```
